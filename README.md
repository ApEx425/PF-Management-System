# PF Management System

A Python-based PF (Provident Fund) Management System built using Object-Oriented Programming (OOP), JSON data storage, and file handling.

## Features

* Create New PF Account
* Generate Unique Account Numbers
* Deposit Funds
* Withdraw Funds
* View Account Details
* Update Account Information
* Delete or Freeze Accounts
* JSON-Based Data Storage
* PIN-Based Authentication

## Technologies Used

* Python
* OOP (Object-Oriented Programming)
* JSON
* File Handling
* Random Module
* Pathlib

## How to Run

1. Clone the repository
2. Open the project folder
3. Run:

```bash
python main.py
```
